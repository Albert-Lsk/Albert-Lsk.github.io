// 关于我

从船舶设计到产品经理，我用4年时间完成这个转变。

华中科技大学船舶专业硕士毕业，研究方向是六轴机器人。在中船605做了2年船舶设计工程师，负责无人艇系统设计和路径规划。那段经历让我理解了复杂系统的工程逻辑，也让我发现自己更享受「定义问题」而非「解决问题」。

2023年转型做产品经理，主导了虎贲一号氢动力无人艇的国际认证，成功拿到LR证书和200万科研资金。现在在跨境电商做硬件产品经理，负责明星产品的全生命周期管理。

同时在探索AI如何改变产品工作——不是做AI产品，而是用AI工具重新定义工作方式。我相信工程背景给了我系统思维，产品经历给了我商业视角，而AI正在成为我的第三个维度。

// 代表作品

虎贲一号氢动力无人艇国际认证

2023-2024

背景：国内氢动力无人船技术缺乏国际认证标准，难以进入全球市场

方案：从0到1完成系统方案设计，与英国劳氏船级社（LR）沟通协调，编制15份技术报告和验证材料，同时以负责人身份完成200万科研项目申报

成果：成功获得LR颁发的AIP证书（Autonomous 1TEU Hydrogen Container Ship），认证时间缩短20%，项目申报获评90分并拿到200万资金支持

系统设计 国际认证 项目管理

SUP气泵产品全生命周期管理

2024-至今

背景：传统SUP气泵充气慢、噪音大，用户体验差，市场竞争激烈

方案：负责公司明星产品SUP气泵的产品定义、功能设计、开发管理和供应链协调，在成本约束下重新设计气泵核心结构和充气逻辑

成果：充气速度提升100%（仅需市场平均时间的一半），显著提升用户满意度和产品竞争力
